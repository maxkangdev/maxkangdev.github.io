## NTP (Network Time Protocol)
>네트워크 상에서 컴퓨터들의 **시간을 동기화**하기 위한 프로토콜. 시간의 정확도를 보장하고, 시간 동기화를 통해 여러 시스템 간의 일관성을 유지하는 데 사용됨 

이런게 왜필요함?? 🤔

1. **분산 시스템의 동기화**
	- 여러 서버와 장치들이 협력해야 하는 시스템에서 시간 불일치가 발생하면 데이터의 일관성이 깨질 수 있음  
    예: 로그 파일의 타임스탬프, 분산 데이터베이스 트랜잭션 등.

2. **보안 및 인증**
	- Kerberos 같은 인증 시스템에서는 정확한 시간이 필요. 시간이 어긋나면 인증 실패가 발생할 수 있음. Certificate도 마찬가지임 (expiration date같은 부분이 정확해야함). 

3. **트러블슈팅**
	- 로그 파일 분석 시 정확한 타임스탬프가 중요함
    예: 서버 장애 발생 시 각 시스템의 시간을 동기화하지 않으면 문제의 원인을 파악하기 어려움.

4. **금융 거래**
    - 주식 거래나 금융 시스템은 밀리초 단위의 정확한 타이밍이 요구됨

5.  **IoT와 네트워크 장치**    
    - IoT 디바이스와 네트워크 장치는 시간 동기화 없이는 효율적인 데이터 전송이나 동작이 어려움.

## 문제
>네트워크 환경에서는 **지연 시간**(latency)와 **네트워크 불균형**(network asymmetry)이라는 문제가 존재함 

### 네트워크 지연 (Latency)
- 클라이언트와 서버간 요청/응답 사이에 데이터가 이동하는 시간 
	- 예를들어, NTP 서버에서 12:00:00 라고 시간을 보내더라도 클라이언트가 해당 정보를 받은 시점은 이미 12:00:50 일수도 있음. 그때 12:00:00으로 세팅해버리면 오차가 생기는 거임. 

### 네트워크 불균형 (Asymmetry)
- 네트워크 트래픽이 클라이언트에서 서버로 가는 경로 (업스트림)와 서버에서 클라이언트로 돌아오는 경로(다운스트림)의 지연 시간이 다를 수 있음 
	- 클라이언트 -> 서버 10ms
	- 서버 -> 클라이언트 20ms 
- 이러한 경우에 왕복시간(RTT)을 단순히 2로 나누는 계산으로는 정확한 시간 동기화를 할 수 없음. 

그래서 더 자세한 계산 방법이 필요함 😩

## 원리 
1. NTP Request (Client -> Server)
	1. T1을 포함하여 메시지를 보냄. 지금 몇시임?
2. Server Response (Server -> Client)
	1. 서버가 요청을 받은 시간 (T2)
	2. 서버가 응답을 보낸 시간 (T3) <- 서버의 현재시각 
	3. 위 정보를 Client 에게 보냄. 지금 12시다~ 
3. Client 수신 
	1. 해당 메시지를 받은 시점을 기록 (T4)
	2. 총 네개의 정보를 가지게 됨 (T1234)

### 계산 
**Round Trip Delay 계산**
$$Delay=(T4−T1)−(T3−T2)$$
**시간 차이(Offset) 계산**
$$ \text{Offset} = \frac{(T2 - T1) + (T3 - T4)}{2} $$
- T1 클라이언트가 요청을 보낸 시간
- T2 서버가 요청을 받은 시간
- T3 서버가 응답을 보낸 시간
- T4 클라이언트가 응답을 받은 시간

## 예제 

**시간 정보**

$$T1=10:00:00.000$$
$$T2=10:00:00.020$$
$$T3=10:00:00.030$$
$$T4=10:00:00.060$$

### Delay 계산 
$$Delay=0.06−0.01=0.05$$
- 네트워크의 왕복 지연 시간은 **50ms**
- 이 값은 클라이언트와 서버 간의 네트워크 속도와 경로를 기반으로 한 네트워크 지연 시간임
### Offset 계산 
$$ \text{Offset} = \frac{0.02 + (-0.03)}{2} = -0.005 $$
- 클라이언트의 시계는 **5ms** 느리다는 것을 알 수 있음
- 클라이언트가 서버의 시간에 맞추려면, 5ms만큼 시간을 앞당겨야 함 
### 동기화 
클라이언트의 시간은 Offset을 더해서 보정함 
$$T4+Offset=10:00:00.060−0.005초=10:00:00.055$$
