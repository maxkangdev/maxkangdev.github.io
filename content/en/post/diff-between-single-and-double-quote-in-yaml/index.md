## Double Quote 

In order to use `\`, you need to add `\`  
`“\\”` -> `\`
`“\””` -> `”`

## Single Quote

You can use `\` and `”` freely, but you have to add `’` to use `’`
`’\\’` -> `\\`
`’”’` -> `”`
`’’` -> `’`
